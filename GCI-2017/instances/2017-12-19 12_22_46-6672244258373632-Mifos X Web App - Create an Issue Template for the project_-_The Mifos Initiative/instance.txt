Title: Mifos X Web App - Create an Issue Template for the project
Description: ## Description

We should create an Issue Template for the Mifos X Web App project. The template should ask the following:

1. Description
2. Steps to reproduce
3. Expected behaviour
4. Observed behaviour
5. Settings
  1. Mifos X version
  2. Browser used
6. Screenshots, if any

We should place the same in `.github` folder under the root directory.
