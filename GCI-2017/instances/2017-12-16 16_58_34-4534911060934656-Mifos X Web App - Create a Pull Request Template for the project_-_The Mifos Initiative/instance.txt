Title: Mifos X Web App - Create a Pull Request Template for the project
Description: ## Description

We should create a Pull Request Template for the Mifos X Web App project. The template should ask the following:

1. Description
2. Related Issues and Discussions
3. Checklist
  1. Whether contributor has read and understood the [contribution guidelines](https://github.com/openMF/community-app/blob/develop/Contributing.md).
  2. Whether [validation](https://github.com/openMF/community-app#validate-js-and-html-files) and [tests cases](https://github.com/openMF/community-app#running-the-tests) passed locally.
4. People to notify

We should place the same in `.github` folder under the root directory.
